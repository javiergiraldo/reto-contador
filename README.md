# Reto 1: Reto Contador

## Descripción

Contador interactivo que permite a los usuarios incrementar, decrementar y resetear un valor numérico, al igual que apagar."

## Objetivo

Demostración del uso de `useState` para la gestión del estado, manejo de eventos (`onClick`) y aplicación de estilos con Tailwind CSS."

## Funcionalidades

Lista las funcionalidades principales del reto. Por ejemplo:

* Incrementar el contador.
* Decrementar el contador.
* Resetear el contador a cero.
* Apagar.

## Tecnologías Utilizadas

* React
* Tailwind CSS

## Cómo Ejecutar localmente

Para probar este reto:

1.  Clona el repositorio.
2.  Navega a la carpeta del reto (`reto-contador).
3.  Ejecuta `npm install`.
4.  Ejecuta `npm run dev`.
